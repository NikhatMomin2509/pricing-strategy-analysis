How the t-Test Works

1

State Hypotheses

H₀: μ = μ₀ (sales unchanged). H₁: μ > μ₀ (sales increased). We test whether data provides enough evidence to reject H₀.

2

Calculate t-Statistic

t = (x̄ − μ₀) / (s / √n). Measures how many standard errors the sample mean is from the expected mean.

3

Find p-Value

The probability of observing this extreme a result if H₀ were true. One-tailed: p = P(T > t). We use α = 0.05.

4

Make Decision

If p < 0.05 → reject H₀ → price reduction significantly increased sales. If p ≥ 0.05 → insufficient evidence.
